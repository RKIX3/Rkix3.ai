# RKIX3 Architecture

RKIX3 is organized as a Vite React app:

- `src/components`: reusable UI atoms/blocks.
- `src/pages`: top-level product pages and center-specific content.
- `src/data`: static project catalog, templates, tasks and integrations.
- `src/services`: browser-safe service wrappers.
- `src/hooks`: reusable state hooks.
- `src/store`: local store helpers.
- `src/utils`: formatting, exporting, parsing and constants.
